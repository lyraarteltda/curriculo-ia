Configure suas chaves de API

Suas chaves ficam salvas apenas no seu navegador. Nunca são enviadas para nossos servidores.

Como consigo uma chave de API?

Você precisa de apenas uma destas chaves para usar a ferramenta:

OpenAI: Acesse platform.openai.com/api-keys e crie uma nova chave.

OpenRouter: Acesse openrouter.ai/keys e crie uma chave — uma só chave dá acesso a vários modelos.

Sua chave fica salva apenas no seu navegador (localStorage). Nunca é enviada aos nossos servidores.
